# OCR 常见误差对照表

仅当怀疑源文档有 OCR 识别误差时读取此文件。

| 正确字符 | 常见 OCR 误识别 | 说明 |
|---------|----------------|------|
| 2023年 | 203年 | 年份数字丢失 |
| 建设 | 建没 | "设"→"没"（形近） |
| 编号 | 编导 | 形近字 |
| 日期 | 曰期 | "日"→"曰"（形近） |
| 联系 | 联杀 | 形近 |
| 规格 | 规模 | 多字尾误增 |
| 型号 | 型马 | 偏旁丢失 |
| 有限 | 右限 | 形近 |
| 技术 | 枝术 | "技"→"枝" |
| 合同 | 合回 | "同"→"回" |
| —（长横） | --（连续短横） | 破折号/长横被拆 |
| ·（间隔号） | ．(句点) | 中点变句点 |

## 检测方法

1. 扫描所有数字年份：`20\d{2}年` 模式，如 `203年`、`205年` 等 3 位数年份
2. 扫描常见形近字：`没`（应检查是否为 `设`）、`枝`（应为 `技`）、`曰`（应为 `日`）、`右`（应为 `有`）
3. 上下文验证：单字替换后在原文语境中是否成立
